Google Sheets 接続設定

Google Apps Script をデプロイして取得したウェブアプリURLを入力してください。

  1. Googleスプレッドシートを新規作成
  2. 拡張機能 → Apps Script → gas-code.gsを貼り付け保存
  3. 左メニュー「サービス」→ Drive API を追加（v3）
  4. デプロイ → 新しいデプロイ → 種類:ウェブアプリ
  5. 実行者:自分 / アクセス:全員
  6. URLをコピーして下に貼り付け
